## build locally

1. clone this
    ```
    git clone 
    ```
2. install all the dependencies
    ```
    npm install
    ```
3. to run the dev server
    ```
    npm run dev
    ```
4. to build and serve
    ```
    npm run build
    npm run deploy
    ```
